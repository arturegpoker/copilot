# Copilot - Single Page Website

Простой и современный одностраничный сайт для демонстрации возможностей GitHub Copilot.

## 🚀 Возможности

- ✨ Современный дизайн с градиентами
- 📱 Полная адаптивность (responsive design)
- ⚡ Быстрая загрузка
- 🎨 Красивые анимации
- 🔒 Чистый и безопасный код

## 📁 Структура файлов

```
.
├── index.html      # Главная страница
├── styles.css      # Стили и дизайн
├── script.js       # Интерактивность
└── README.md       # Документация
```

## 🌐 Развертывание на Cloudflare Pages

### Способ 1: Через веб-интерфейс Cloudflare

1. Перейди на [Cloudflare Pages](https://pages.cloudflare.com/)
2. Нажми "Create a project"
3. Подключи свой GitHub репозиторий `arturegpoker/copilot`
4. Выбери ветку для развертывания
5. Оставь стандартные настройки:
   - **Build command:** (оставить пусто)
   - **Build output directory:** / (корневая папка)
6. Нажми "Save and Deploy"

### Способ 2: Через Wrangler CLI

```bash
# Установи Wrangler
npm install -g wrangler

# Войди в аккаунт Cloudflare
wrangler login

# Разверни сайт
wrangler pages deploy .
```

### Способ 3: Drag & Drop

1. Перейди на [Cloudflare Pages](https://pages.cloudflare.com/)
2. Выбери "Upload assets"
3. Перетащи папку с файлами проекта

## 🎯 После развертывания

После развертывания на Cloudflare Pages тебе будет выдана ссылка вида:
```
https://copilot.pages.dev
```

Ты можешь подключить свой кастомный домен через настройки проекта.

## 🛠️ Локальная разработка

Чтобы тестировать сайт локально:

```bash
# Используй простой HTTP сервер
python -m http.server 8000

# Или если у тебя установлен Node.js:
npx http-server
```

Затем открой `http://localhost:8000` в браузере.

## 📝 Настройка

Для изменения контента отредактируй файлы:

- **index.html** - структура и текст
- **styles.css** - цвета, шрифты, стили
- **script.js** - функциональность
