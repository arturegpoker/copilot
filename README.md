# Vacation Management (Учет отпусков)

Простой бэкенд на FastAPI для учета отпусков, больничных и отпусков за свой счет.

Функционал:
- сотрудники создают запросы на время отсутствия (vacation, sick, unpaid)
- руководители просматривают запросы и одобряют/отклоняют
- можно получить календарь утверждённых отсутствий по месяцу

Запуск локально:
1. Установите зависимости: pip install -r requirements.txt
2. Запустите приложение: uvicorn main:app --reload
3. Откройте документацию: http://127.0.0.1:8000/docs
